# PawTube Project Description

PawTube is an open-source video sharing platform inspired by the classic
2005--2012 YouTube era. It uses Firebase Authentication and Firestore
for account management and video listings, while all videos are embedded
from external hosting platforms such as Vimeo. PawTube follows a
lightweight, community‑moderated model with strict upload guidelines to
ensure safety and originality.

## Key Features

-   Classic old‑YouTube aesthetic\
-   User accounts with Firebase Auth\
-   Embed‑based video uploads stored in Firestore\
-   Automatic listing on the homepage\
-   Channel pages linked to each user\
-   Manual moderation system

## Upload Guidelines Summary

1.  No spams\
2.  No 18+ content (13+ allowed with disclaimer)\
3.  Embeds must be valid video embeds\
4.  No embeds that break the site\
5.  Only upload videos you own or have permission for\
6.  No copyrighted content\
7.  Videos must be functional and watchable\
8.  No scams such as Free Robux\
9.  No exposing other people's channels\
10. Report violations to **pixelpaw-coding@keemail.me**

PawTube is maintained as a community-driven, open-source project where
users can contribute, submit issues, or suggest features.
